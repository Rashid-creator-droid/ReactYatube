# YaTube на React

Переделываю YaTube на React

##



### Как запустить проект:

Клонировать репозиторий и перейти в него в командной строке:

```
https://github.com/Rashid-creator-droid/ReactYatube.git
```

```
cd ya-tube/
```

Установить зависимости:

```
npm i
```

Запустить проект:

```
npm run start
```
